<p align="center">
 <code>godmarc</code> is a TUI <a href="https://datatracker.ietf.org/doc/html/rfc7489">DMARC</a> report analyzer.
</p>

---

### Usage

Place your DMARC report files in `~/.godmarc` and run `godmarc`.

```
go install github.com/huhndev/godmarc@latest
godmarc
```
